# img2bw

Image binarizer


## Installation

Open the terminal, go to the folder of this package and type:

On Ubuntu/Debian/MacOS:

```
pip install img2bw
```


## Usage


To binarize a single image, type:

```
img2bw example.jpg
```

To binarize all the images in a directory, type:

```
img2bw input_dir/ --output output_dir/
```
